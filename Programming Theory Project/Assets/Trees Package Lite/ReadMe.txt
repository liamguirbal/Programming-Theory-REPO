 	-How to Use-

1.  Import the package into your Unity project via the Asset Store or Package Manager.
2.  Navigate to the `Assets/Cozy Kitchen/Prefabs` folder.
3.  Drag and drop the desired sofa prefab into your scene.
4.  For URP/HDRP: If your project uses URP or HDRP, you may need to upgrade the materials by going to `Edit > Render Pipeline > Universal Render Pipeline/High Definition Render Pipeline > Upgrade Project Materials to...`. The package also includes ready-to-use material versions for URP/HDRP.
